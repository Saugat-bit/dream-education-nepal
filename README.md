# Dream Education Nepal

Animated TypeScript website for Dream Education Nepal, built with Vite and GSAP.

## Folder Structure

- `src/assets/logos` - client logo files, partner logos, and favicon source assets.
- `src/assets/videos` - animated hero videos, section loops, and compressed web video exports.
- `src/assets/images` - photos, posters, texture images, and social preview artwork.
- `src/animations` - reusable GSAP timelines and scroll animation helpers.
- `src/components` - reusable UI sections and page components as the project grows.
- `src/data` - structured content for English and Japanese copy.
- `src/i18n` - future translation utilities if the bilingual system grows beyond static content.
- `src/styles` - global styles and section styling.

## Commands

```bash
npm run dev
npm run build
npm run preview
```
